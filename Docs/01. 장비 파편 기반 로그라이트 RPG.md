# 장비 파편 기반 로그라이트 RPG
## 파편 · 분해 · 제작 · 각성 · 시즌 경쟁 · 거래소 · NFT · 현금 BM
---
## 1. 게임 핵심 철학
- 장르: 로그라이트 RPG
- 핵심 루프  
    **파밍 → 파편 조합 → 장비 제작 → 각성 선택 → 시즌 경쟁 → 거래**
- 경제 구조: **플레이어 주도 거래소**
- 고점 콘텐츠: **각성 장비 + 시즌 경쟁 기록**
### 설계 원칙
- NFT 없이 모든 콘텐츠 플레이 가능
- NFT는 **각성 장비의 신뢰·유일성 증명 용도**
- 확률 요소는 **제작 이전까지만**
- 제작 이후 결과는 **100% 사전 공개**

## 2. 핵심 플레이 루프
```
던전 플레이
 → 장비 파편 획득
 → 파편 조합 → 장비 제작
 → 강화
 → 각성 여부 선택
 → 시즌 경쟁 콘텐츠
 → 거래소 활용
 → 시즌 종료 및 보상 분배
```
- 각성은 목표이지 필수 아님
- 미각성 상태로 모든 일반 콘텐츠 가능
- 차단되는 건 **공식 기록 등록/보상 자격**뿐

## 3. 장비 파편 시스템
### 3.1 파편의 역할

|요소|설명|
|---|---|
|기본 스탯|완성 장비 수치 구성|
|잠재 범위|각성 시 최대치 근거|
|고유 스킬|장비 스킬 후보|
- 파편 자체는 NFT 아님
- 거래소 유동성의 핵심 자산
### 3.2 완성 장비 스탯 결정 규칙
- 장비 스탯은 **사용된 모든 파편 정보로 확정**
- 랜덤 / 리롤 / 변경 없음
```
완성 장비 스탯 =
Σ(파편 기본 스탯)
+ 파편 조합 보정
+ 잠재 최대치 계산
```
제작 순간에 다음 정보가 모두 확정되고 UI에 노출된다.
- 현재 스탯
- 각성 시 최종 스탯
- 각성 특수 효과
- 스킬 선택 후보

## 4. 장비 제작 & 스킬 선택
### 4.1 제작 규칙
- **코어 파편 1 + 자유 파편 2** 조합으로 장비 제작
```
코어(무기) + 자유 + 자유 → 무기
코어(방어구) + 자유 + 자유 → 방어구
코어(장신구) + 자유 + 자유 → 장신구
```
#### 파편 역할
- 코어 파편
    - 장비 타입 결정(무기/방어구/장신구)
    - 주요 스탯 결정
- 자유 파편
    - 보조 스탯
    - 스킬 후보 제공
**효과**
- 제작 결과 예측 가능
- 거래소 유동성 유지
- 빌드 다양성 확보
### 4.2 시즌별 규칙 변주
- 시즌마다 **기본 규칙은 고정** 바뀌는 건 **보너스 규칙(메타 방향)만**.
#### 시즌 규칙 예시
- S1: 기본 시즌
    - 추가 규칙 없음 (학습/시장 안정)
- S2: 타입 통일 보너스
    - `코어 타입 = 자유 타입 ≥ 1` → 잠재 최대치 +α
- S3: 혼합 장려
    - `자유 2개가 서로 다른 타입` → 스킬 계수 보너스
- S4: 코어 강화
    - `시즌 지정 코어 사용` → 주요 스탯 가중치 증가
- S5: 자유 파편 특화
    - `자유 2개 타입 동일` → 보조 스탯 효율 증가
### 4.3 스킬 선택
- 파편마다 스킬 1종 제공
- 완성 장비는 **그중 1종만 선택**
```
파편 A → 스킬 A
파편 B → 스킬 B
파편 C → 스킬 C
→ 최종 선택: A / B / C 중 1
```
- 제작 완료 시 1회 선택
- 이후 변경 불가

## 5. 파편 분해 · 재제조
### 5.1 분해 결과
- 파편 1개 분해 → **분해 재료 A xN**
- N은 등급/티어로 스케일(밸런스 파라미터)
### 5.2 스킬 추출
- 파편 분해 시 추가로 **스킬 에센스** 획득
- **고정 1개**(파편이 가진 스킬ID 1개를 그대로 에센스로 환원)
### 5.3 거래소 미판매 만료 보너스
- 거래소 등록 후 **T일 경과**하여 **미판매 만료/회수 플래그**가 붙은 파편 분해 시:
    - `재료 A xN` + `재료 A 보너스 x(N*B%)`
- 보너스 적용 범위:
    - **재료 A에만 적용**
    - **스킬 에센스에는 미적용**
    - 보너스는 **계정/주간 캡** 또는 **점감**

## 6. 파편 제조
### 6.1 기본 재제조
- `재료 A xM` 소비 → 파편 1개 생성
### 6.2 파편 카테고리 선택
- 재제조 UI에서 **생성 파편 카테고리를 선택** 가능
- 최소 카테고리:
    - **무기 / 방어구 / 장신구 / 공용(전체 랜덤)**
- 비용 차등:
    - 공용 랜덤: `A xM`
    - 카테고리 선택: `A x(M+Δ)`  
### 6.3 스킬 확정 제작
- 재제조 시 옵션
    - **에센스 투입 O**: 해당 스킬로 **확정된 파편** 생성
    - **에센스 투입 X**: 스킬은 **랜덤**(카테고리/티어 룰에 따름)
- **주간/계정 제작 횟수 제약** 적용

## 7. 미각성 장비 분해
### 7.1 목적
- 실수 복구/인벤 정리 UX 개선.
- 단, 경제 붕괴 위험이 커서 **반드시 손실/제약 기반**으로 설계한다.
### 7.2 허용 범위
- **각성 안 된 장비만 분해 가능**
### 7.3 반환 규칙
- 장비 분해 결과는 **파편 반환 금지**
- 반환은 **분해 재료 A만**
- 반환량은 “투입 파편 3개를 그냥 분해했을 때 총합” 대비 **30~60% 수준**
### 7.4 에센스 규칙
- 장비 분해에서 **스킬 에센스는 지급하지 않음**
- 에센스는 **파편 분해에서만** 얻는다
### 7.5 추가 안전장치
- 장비 분해 **일일/주간 횟수 제한**
- 분해 UI에서 **반환 A 수치 확정 표시**

## 9. 각성 시스템
### 9.1 각성 정의
각성 = **잠재 최대 상태로 영구 고정**
- 잠재 최대 스탯 적용
- 특수 효과 활성화
- 상태 변경 완전 불가
- 시즌 경쟁 기록 자격 획득
### 9.2 각성 조건
- 장비 최대 강화
- 각성 비용 지불
- 실패 없음
### 9.3 각성 특수 효과 성격
- 단순 수치 상승 아님
- 플레이 스타일 변화 중심
예:
- 특정 조건 자원 회수
- 특정 행동 강화
- 조건부 쿨타임 감소

## 11. 시즌 기반 경쟁 콘텐츠
### 11.1 성격
- 시즌제 PvE 기록 경쟁
- 예:
	- 던전 타임어택
	- 웨이브 생존 점수
	- 조건부 챌린지 클리어 점수
### 11.2 기록 조건
```
모든 착용 장비 각성
→ 공식 기록 등록
→ 시즌 보상 분배 대상
```
- 미각성 장비 사용:
    - 플레이 가능
    - 기록 미등록
    - 보상 없음
    - **공식 기록 자격만 제한**
### 11.3 시즌 종료
- 랭킹 동결
- 기록 갱신 불가
- 각성 장비 성능 유지와 파밍에 도움을 주는 특수 효과 부여
- **각성 장비의 유효 시즌을 감소시킴**
### 11.4 시즌 종료 후 각성 장비의 의미
- 성능 유지
- 일반 콘텐츠 사용 가능
- 거래 가능
- 장비 파편 파밍에 도움을 주는 특수 효과 부여
	- 예: 능력치 또는 드랍율 상승 등
- **제작 시즌의 +n 시즌까지만 시즌 기록에 참여 가능**

## 12. 던전 입장 시스템
### 12.1 던전 유형별 입장 방식

| 던전 종류    | 입장 방식         | 설계 의도                  |
| -------- | ------------- | ---------------------- |
| 일반 파밍 던전 | 스태미나(에너지) 소모  | 기본 플레이 루프 제어, 신규 유저 친화 |
| 고급 파편 던전 | 입장권 + EC 소모   | 고급 자원 공급량 통제, 파편 가치 유지 |
| 경쟁 콘텐츠   | 무료 입장 + 기록 제한 | 연습 자유 보장, 랭킹 악용 방지     |
| 시즌 챌린지   | 주간/시즌 횟수 제한   | 공정한 기록 경쟁, 시즌 관리 용이    |
| 연습 모드    | 무제한 입장 / 무보상  | 숙련·빌드 테스트 전용           |
### 12.2 세부 규칙 정리
#### 일반 파밍 던전
- 입장 시 스태미나 소모
- 보상: 기본 파편, EC
- 경쟁 요소 없음
#### 고급 파편 던전
- 전용 입장권 + EC 소모
- 고급 파편 드랍
- 과도한 반복 파밍 방지 목적
#### 경쟁 콘텐츠 던전
- 입장 비용 없음
- 플레이는 무제한
- 공식 기록 및 보상은 제한 횟수 내에서만 등록
- 미각성 장비 사용 가능하나 기록 미등록
#### 시즌 챌린지 던전
- 계정 단위 주간 또는 시즌 입장 횟수 제한
- 실패해도 횟수 차감
- 모든 착용 장비 각성 필수
#### 연습 모드
- 모든 경쟁 던전에서 파생
- 보상 없음
- 기록 미등록

## 13. 거래소 & 재화 구조
### 13.1 재화 분리 원칙

| 구분    | 명칭  | 용도       |
| ----- | --- | -------- |
| 거래 재화 | MT  | 거래소 전용   |
| 성장 재화 | EC  | 제작·강화·각성 |
### 13.2 MT (마켓 토큰)
**성격**
- 현금 구매 불가
- 거래소 외 사용 불가
**생성 (Source)**
1. 경쟁 콘텐츠 시즌 보상
	- 시즌 종료 시
	- 랭킹 상위 플레이어에게 분배
	- 재원: 거래 수수료 풀
	```
	거래 발생
	→ 수수료 적립
	→ 시즌 종료
	→ 상위 기록자에게 MT 분배
	```
2. 일일 경쟁 챌린지 소량 보상
	- 기록 등록 실패해도  
	- 참여 보상으로 소량 지급
**사용 (Sink)**
- 장비 파편 / 미각성 장비 / 각성 장비(NFT) 구매
**소각 (Burn)**
```
거래 수수료 10%
- 6% → 시즌 보상 풀
- 4% → 즉시 소각
```
- 고가 거래 추가 소각 적용
### 13.3 EC (에너지 크레딧)
- 계정 귀속
- 거래 불가
**생성 (Source)**
- 던전
- 일일 / 주간 퀘스트
**사용 및 제거 (Sink and Burn)**
- 장비 제작
- 장비 강화
- 장비 각성
    → 사용 즉시 소멸

## 15. 현금 유입 비즈니스 모델
### 15.1 절대 금지
- MT 현금 구매 ❌
- 각성 / 랭킹 / 기록 접근 ❌
- 제작 결과 변경 ❌
### 15.2 허용 BM 범위
#### 편의성
- 제작 슬롯 확장
- 창고 / 거래소 슬롯 확장
#### 시간 단축
- 제작 대기 시간 감소
- 일일 플레이 보조 패키지  
    (EC 소량, MT 없음)
#### 시즌 패스
- 무료: EC, 파편, 일부 코스메틱
- 유료: 코스메틱, 제작 가속, 시즌 테마 연출  
    ❌ MT / 각성 / 경쟁 보상
#### 코스메틱
- 캐릭터 외형
- 장비 외형 오버레이
- UI 테마
